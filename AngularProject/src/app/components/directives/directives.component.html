DIRECTIVES

A directive provides a way of invoking Angular functionality within an HTML file. They act as syntax that can be used in the HTML format without breaking our front end.

*ngIf

What does the asterisk(*) in front of the ngIf mean?

Any directive that has an asterisk(*) is a "structural directive". This means that the directive in question will actually manipulate the HTML directly. This is, it will be capable of physically deleting or adding HTML on the spot for us.

*ngFor

Since ngFor is another directive that will change the actual HTML directly, we need the asterisk to signify that it is a structral directive.

Attribute Directives

Attribute directives are directives that are applied as attributes in the elements themsevles, like structural directives. However, these directives typically do NOT aim to change the actual structure of the HTML, but rather change the style.
